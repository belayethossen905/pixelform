# Pixelform — UI UX Design Agency Landing Page

A fully self-contained, single-file landing page ready to deploy on Vercel, Netlify, GitHub Pages, or any static host.

---

## 🚀 Deploy on Vercel (Recommended)

### Option A — Vercel Dashboard (No code needed)
1. Go to [vercel.com](https://vercel.com) and sign up / log in
2. Click **"Add New Project"**
3. Choose **"Upload"** and drag this entire folder in
4. Click **Deploy** — your site is live in ~30 seconds!

### Option B — Vercel CLI
```bash
npm install -g vercel
cd pixelform
vercel
```
Follow the prompts and your site will be live instantly.

---

## 🌐 Deploy on Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop this entire folder onto the Netlify dashboard
3. Done — live URL generated instantly!

---

## 📄 Deploy on GitHub Pages
1. Push this folder to a GitHub repository
2. Go to Settings → Pages
3. Set source to `main` branch / root
4. Your site will be live at `https://yourusername.github.io/repo-name`

---

## ✏️ Customization Checklist
Before going live, update these in `index.html`:
- [ ] Agency name: `Pixelform` → your brand
- [ ] Email: `hello@pixelform.co` → your email
- [ ] Location & availability text
- [ ] Social media links in the footer
- [ ] Pricing numbers
- [ ] Contact form (connect to Formspree, Netlify Forms, or your backend)
